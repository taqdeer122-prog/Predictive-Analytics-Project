# 💼 Predictive Analytics for Salary Estimation using SPSS Modeler and Python

## 📌 Overview
This project focuses on building a **salary prediction system** using both **IBM SPSS Modeler** and **Python-based machine learning techniques**. The goal is to analyze employee-related factors such as experience, education level, and job role to accurately predict salary outcomes.

The project follows a structured **CRISP-DM methodology**, ensuring a systematic approach from data understanding to model evaluation.

---

## 🎯 Objectives
- Predict salary based on employee attributes  
- Identify key factors influencing salary  
- Compare multiple machine learning models  
- Generate insights for HR and business decision-making  

---

## 📊 Dataset Description
The dataset used (*salary_data_cleaned.csv*) contains structured employee data including:
- Experience (years)
- Education level
- Job role / designation
- Department (if available)
- Salary (target variable)

The dataset is preprocessed and cleaned to ensure consistency and accuracy.

## 📈 Results & Findings
- Experience has the strongest impact on salary  
- Education level significantly affects salary growth  
- Certain job roles offer higher compensation  
- Linear Regression provided the best performance  
- Model achieved good accuracy with low prediction error  

---

## 💡 Business Impact
- Helps in designing fair salary structures  
- Improves recruitment and compensation strategies  
- Enables salary benchmarking  
- Reduces bias in salary decisions  
- Supports data-driven HR decision-making
